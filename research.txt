# NPM Supply Chain Attack Detection System in python

## Overview

This project presents a research-grade, modular pipeline for detecting supply chain attacks in NPM packages using a combination of static code analysis, metadata analysis, dependency graph analysis, and advanced feature fusion techniques.

The system leverages machine learning with LightGBM to classify packages as malicious or benign.

---

## Objective

The goal of this project is to design and implement a scalable, reproducible, and extensible detection system capable of identifying:

* Malicious package injections
* Obfuscated code execution
* Suspicious dependency chains
* Zero-day supply chain attack patterns

## Module Description

### Data Ingestion

* Collects NPM packages via API and datasets
* Extracts source code, package.json, and dependency trees

### Static Analysis

* Detects dangerous APIs (eval, child_process, etc.)
* Identifies obfuscation patterns
* Performs entropy and suspicious logic analysis

### Metadata Analysis

* Extracts author reputation
* Detects version anomalies
* Analyzes install scripts and update behavior

### Dependency Graph Analysis

* Builds dependency graphs
* Extracts centrality, depth, and anomaly features

### Feature Engineering

* Normalization and encoding
* Feature selection
* Missing value handling

### Feature Fusion

* Early fusion (concatenation)
* Late fusion (ensemble)
* Hybrid fusion strategies

### Model Training

* Uses LightGBM
* Includes cross-validation and hyperparameter tuning
* Handles imbalanced datasets (SMOTE)

### Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC
* False Positive Rate

### Reporting & Output

* Trained model files
* Evaluation reports
* Feature importance (SHAP)
* Logs

## How to Run

1. Clone Repository
   git clone <repo_url>
   cd npm_supply_chain_detector

2. Install Dependencies
   pip install -r requirements.txt

3. Run Full Pipeline
   python main.py --stage full

4. Run Specific Stage
   python main.py --stage train
   python main.py --stage evaluate
